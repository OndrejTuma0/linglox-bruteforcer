# Linglox Bruteforcer
A bruteforcer for the roblox game **Linglox**, guesses all common words using a first letter, last letter and word length.
Use the two word file to bruteforce two word prompts. (Will take longer, depending on how big word list is.)

## Controls
- **Get Words** to fetch words upon filling in parameters
- **Show Words** to show the current word list that it will bruteforce
- **F1** to start bruteforcing
- **F2** to stop when done
